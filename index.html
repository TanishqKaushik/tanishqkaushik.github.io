ML Enhanced Predictive Modeling for Diabetes Diagnosis

This project aims to revolutionize diabetes diagnosis using data science. By meticulously preparing and analyzing health data, the project unveils vital insights. It classifies individuals based on their BMI, uncovers demographic patterns, and highlights the influence of age, gender, and BMI on diabetes risk. Cluster Analysis of Credit Card Users

This project dives into an 8600-row, 17-feature credit card dataset with a comprehensive cluster analysis. Through systematic data preprocessing, including careful choices between normalization and standardization, it enhances analytical precision. Utilizing the K-means algorithm, the project maps SSE across diverse cluster counts and identifies the optimal number with an Elbow method plot. The outcome is a visually compelling scatter plot highlighting distinct clusters for effective data interpretation and insight extraction.

Car Price Prediction Using Classification Model

This project crafts a classification algorithm for car price forecasting, utilizing Decision Tree and Random Forest models. With astute data preprocessing encompassing cleaning, transformation, and categorical feature encoding, it primes the dataset. A 70-30 train-test split assesses the Decision Tree classifier's performance, employing both GINI and Entropy criteria. The project also offers fascinating visualizations of the Decision Tree's decision-making process. Ultimately, the Random Forest classifier enhances accuracy scores, ensuring precise car price predictions.
